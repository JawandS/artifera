Social Network Solution
Problem: students are not getting the most effective learning materials and teachers don’t have the time or incentive to implement new ones
A bounty system in a social where the top teachers/schools get money for their artifacts and randomly selected users get funding for their classroom as well
Funding sources
General nonprofits for education
Specific AI companies looking for AI products hackathon 
Use the network as training data for AI products (have a paid opt-out)
Security: require teacher authentication, 2FA, minimize conflict of interest in hackathons 
Initial problems: 
AI can improve education, but how do you make it frictionless to use?
The communities that can most benefit from AI tooling don’t have the time or resources to leverage it

Constraint: AI is moving fast, and static resources quickly become outdated 
Creates a lot of fatigue among educators who don’t feel like they can keep up
Speed of everything continues to increase

Under resourced schools are a global problem
Question: how could AI address the student / teacher ratio?
Is it only feasible via resource optimization?
Can consider educating people outside the classroom as well
Can we make it easier for the teacher to manage the classroom somehow?
Can we create “digital twins” of teachers to augment the number of educators	
Initial idea:
Creating a course / landing page for educators to leverage AI 
Better informing teacher on how to user AI
Creating specific knowledge bases for each class (e.g. Claude projects, NotebookLMs)
Aggregate existing tutorials / information to help teachers 
Especially in non-English speaking countries 
Key innovation: is making existing tools more practical to use 
Create a chatbot that interviews the teacher and recommends resources based on their answers
Creates a custom course for the teacher to learn more
Personalizing content with AI but make it so that one device is needed instead of multiple
Creating a social media network for teachers focused on modern practices (especially AI)
Create a bounty system where the teachers with the most upvotes get monetary prizes to fund their classrooms (be careful with fake upvotes, similar issues to other social media networks)
Initially seed the network with team generated content, focus on peer-to-peer long-term
Emphasizing concrete artifacts as opposed to generalized advice 
Repository of artifacts, where the best ones result in monetary rewards for the teachers  
Examples: grading rubrics, class projects, prompts, notebooks with content, vibe coded websites 
Primary issue: finite set of artifacts among teachers that may get exhausted
Offset by new teachers 
Offset by financial motive to innovate on new products 
Funding
Type 1: general education donations go to this programs 
Type 2: hackathons, specific companies offer tracks (e.g. use OpenAI and get the top vote to get XX dollars)
